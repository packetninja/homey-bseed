## What This Project Contains

This is a **derivative work** containing:
- 3 BSEED wall switch drivers from Universal Tuya Zigbee
- Subset of the lib/ framework (optimized from 5.2MB to 2.9MB)
- Development-focused structure for debugging with `homey app run`

All core functionality, driver implementations, and library code are the work of **Dylan Rajasekaram**, based on foundations by **Johan Bendz**.
